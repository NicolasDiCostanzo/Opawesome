## 📌 About
Opawesome is a simple, customizable meme creator built with Vue.js. Create, share, and laugh with your own memes—all in your browser!

**Features:**
- Drag-and-drop meme templates
- Custom text and styling
- Export as PNG/JPEG
- Open-source and free forever
