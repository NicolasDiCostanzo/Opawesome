## 📌 About
Opawesome is a tool for creating opossum-based. I found it challenging to make opossum-style memes so I decided to create my own tool. It was made with lot of love, passion, frustration, anger, and a subtle touch of nihilism. 
You can help me make this project live beyond my room by either contributing or making stuff with it and sharing it!

**Features:**
- Make memes with opossums on them, duh!

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or later)
- npm

### Project Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/nicolasdicostanzo/opawesome.git
   cd opawesome
   
2. Install dependencies:
   ```bash
   npm install

3. Development
   Run the dev server:
   ```bash
   npm run serve

4. Do your stuff buddy 💪
